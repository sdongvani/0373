# Mi proyecto

crear carpeta por terminal:
`mkdir <>`

crear archivos por terminal:
`touch <>`

### Pasos a seguir para subir los cambios a github:

Ver el estado de los cambios:
`git status`

Añadir los archivos nuevos:
`git add .`

O añadir archivos específicos:
`git add nuevo-archivo.txt README.md`

Hacer commit de los cambios:
`git commit -m "Agregar nuevos archivos y estructura"`

Subir a GitHub:
`git push origin main`
